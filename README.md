# RMA Returns Monitor — customer download

This repository hosts a single customer deliverable: the **RMA Returns Monitor** managed
solution, packaged for import into a Power Platform environment.

- **Download:** see the [latest Release](../../releases/latest) → `RMAReturnsMonitor_managed.zip`
- **Download page:** https://billwhalenmsft.github.io/rma-returns-monitor/

## Import (≈5 minutes)

1. Download `RMAReturnsMonitor_managed.zip` from the Release.
2. Go to `make.powerapps.com` and select your target environment.
3. **Solutions → Import solution**, browse to the `.zip`, then **Next → Import**.
4. Open the solution to confirm its components loaded.
