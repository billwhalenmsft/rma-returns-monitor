Import in ~5 minutes

  1. Download RMAReturnsMonitor_managed.zip using the button above.
  2. Go to make.powerapps.com and select your target environment (top-right).
  3. In the left nav choose Solutions → Import solution.
  4. Browse to the downloaded .zip, then Next → Import.
  5. When the import completes, open the solution to confirm its components loaded.
